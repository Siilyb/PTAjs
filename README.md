## 快速上手教程
1.安装浏览器插件 Tampermonkey (油猴)。

2在下面链接中下载，或在此直接下载

(https://greasyfork.org/zh-CN/scripts/560207-pta-pintia-%E5%AD%A6%E4%B9%A0%E5%8A%A9%E6%89%8B-%E5%85%A8%E8%83%BD%E7%89%881-0)

3.进入 PTA 题目页面，点击出现的"开始答题”按钮即可。
